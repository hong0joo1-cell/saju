# 명품 사주 분석 - 전문가가 전하는 당신의 운명

## 프로젝트 개요
한국 전통 명리학 기반의 AI 사주 분석 웹 서비스
- 테마: 화이트 베이스 + 황토색(#C8A96E, #D4A853) 포인트
- 폰트: Noto Serif KR + 포인트 폰트 조합
- 반응형 레이아웃 및 모바일 최적화

## 기술 스택
- Frontend: React 19 + Tailwind CSS 4 + TypeScript
- Backend: Express 4 + tRPC 11
- Database: MySQL/TiDB
- LLM: Manus Built-in LLM API
- 차트: Recharts (레이더 차트, 오행 시각화)
- 인증: Manus OAuth

## 디자인 가이드
- 주색상: #FFFFFF (화이트)
- 포인트색: #C8A96E, #D4A853 (황토색)
- 폰트: Noto Serif KR (기본), 포인트 폰트 (제목)
- 스타일: 전통적이고 격조 있는 분위기
- 레이아웃: 대칭적이고 여유있는 공간 활용

## 외부 API/라이브러리
- 사주 계산: 천간지지 알고리즘 (자체 구현)
- 오행 분석: 오행 상생상극 로직 (자체 구현)
- 차트: Recharts RadarChart
- 이미지 생성: 캔버스 기반 카드 생성 (자체 구현)
